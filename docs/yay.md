#### Install Yay
* `sudo pacman --noconfirm -S git`
* `git clone https://aur.archlinux.git/yay.git`
* `cd yay`
* `makepkg -si`

#### Install linux-macbook
* `sudo yay -S linux-macbook`
* `sudo yay -S kblight'
